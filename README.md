## 虛擬機
>my_env\Scripts\activate #進入虛擬機環境
>(my_env)>deactivate #退出虛擬機環境

## 啟動server
>進入虛擬機環境
>python manage.py migrate #看起來只需要下第一次
>python manage.py runserver #啟動server 預設為http://127.0.0.1:8000/
